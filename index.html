Project Overview

AirSense is a hardware‑hacked air quality monitor upgraded with WiFi connectivity and smart home integration. It exposes environmental data such as particulate matter levels, ambient light, and device diagnostics to a real‑time dashboard for monitoring and automation.

Key Features

Real‑Time Air Monitoring

Continuously measures particulate matter and sends updates to the smart home dashboard.

Ambient Light Detection

Detects room brightness and enables automation such as dashboard dimming.

Device Diagnostics

Monitors uptime, memory usage, WiFi signal strength, and system health.

Smart Home Automation

Trigger automations like ventilation alerts when air quality drops.

Hardware Modification

The internal particulate sensor was tapped through UART and connected to a WiFi microcontroller for data streaming.

Dashboard Visualization

Environmental data visualized through a smart home dashboard interface.
