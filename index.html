Explore Features

Project Overview

AirSense is a modified IKEA Vindriktning air quality monitor converted into a fully connected IoT device using ESPHome. The project exposes particulate matter readings, ambient brightness levels, device diagnostics and WiFi metrics directly to a smart home dashboard.

Key Features

PM2.5 Monitoring

Reads particulate concentration from the internal PM1006 sensor using UART.

Ambient Brightness

ADC sensor measures environmental light and converts it to brightness percentage.

WiFi Telemetry

Reports WiFi strength, uptime, heap memory and device status.

OTA Firmware

Update firmware remotely without opening the device again.

Hardware Modification

The original sensor was opened and an ESP8266 D1 Mini was integrated with the internal PM1006 air sensor.

Pin Connection Table
