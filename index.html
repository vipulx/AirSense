Project Overview

AirSense transforms a basic air quality monitor into a fully connected smart device. Using an ESP8266 and custom firmware, the device streams environmental data directly to a smart home dashboard with real‑time monitoring and automation capabilities.

Key Features

Air Quality Monitoring

Real‑time PM2.5 monitoring using the internal sensor with smart home integration.

WiFi Connectivity

Wireless integration enabling live data dashboards and remote firmware updates.

Ambient Light Detection

Built‑in brightness sensing for automation and dashboard brightness control.

Device Diagnostics

Tracks uptime, memory usage, WiFi signal strength, and device health.

Hardware Hack

ESP8266 integrated inside the air sensor with UART data tapping.

Smart Home Dashboard

Live data visualized on the home automation dashboard.
